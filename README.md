# protea-slackbot

## 起動

### 起動コマンド
```$ docker-compose up ```


### .env file
```
# slackボットの情報
SLACK_BOT_TOKEN
SLACK_APP_TOKEN

# APIサーバのドメイン名またはIPアドレス
API_SERVER_DOMAIN

# 定期実行によるメッセージの送信先
CHANNEL_ID
```
